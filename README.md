# Focus Flow - シンプル自己管理Webアプリ

## 📖 作品概要
「集中（Focus）して、流れるように（Flow）タスクをこなす」をコンセプトにした、自己管理Webアプリケーションです。
複雑な機能を削ぎ落とし、直感的な操作で日々のタスク管理を行えるよう設計しました。

## 🔗 デモサイト
[デモサイトはこちら](https://kiohero32491.github.io/focus-flow/)

## 🎯 ターゲット
自身を含めた、日々の時間管理に課題を感じている学生・社会人

## 💡 制作意図
私自身が抱えていた「時間配分がうまくいかない」という課題を解決するために開発しました。
多機能なアプリは管理自体が手間になることがあるため、「タスクを入れる→実行する」というシンプルな体験を最優先し、ユーザーが目の前の作業に没頭できるデザインを目指しました。

## 🛠 コーディング・運用のこだわり

### 1. AIを「技術メンター」として活用した開発
本アプリは、生成AI（Gemini）をコーディングの補助ツールとして導入しました。
単にコードを生成させるだけでなく、**「なぜこのコードになるのか」「エラーの原因は何か」**をAIに解説させながら実装することで、JavaScriptへの理解を深めつつ、短期間での開発を実現しました。

### 2. スマートフォンでの操作性を追求
外出先や移動中でもサッと確認できるよう、モバイルファーストでの設計を行いました。
ボタンの押しやすさや文字サイズ、レスポンシブ時の表示崩れがないかを徹底的に検証し、ストレスのないUIに仕上げています。

### 3. GitHub PagesによるWeb公開
実務での運用を想定し、GitHub Pagesを用いてWeb上へのデプロイ（公開）を行いました。
PC・スマートフォンの両方ですぐに動作確認ができる環境を整えています。
